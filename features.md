# EDwIC — Опис Можливостей та Функціоналу (v3.3)

EDwIC (ESP8266 Telegram Bot) — це потужна система моніторингу та автоматизації мікроклімату, розроблена для забезпечення максимального контролю через Telegram та сучасний веб-інтерфейс.

## 🌡️ Моніторинг та Сенсори
- **Підтримка датчиків:** Температура, вологість та атмосферний тиск (BME280, HTU21D).
- **Висока точність:** Показники відображаються з точністю до десятих.
- **Живий статус:** Дані оновлюються в реальному часі на OLED-дисплеї, в Telegram та у веб-дашборді.

## 🤖 Автоматизація та Режими Керування
1. **Ручний режим:** Керування через фізичну кнопку, Telegram (inline кнопки) або веб-інтерфейс.
2. **Автоматика по датчику (Термостат):** Увімкнення/вимкнення реле залежно від цільової температури та гістерезису.
3. **Автоматика по часу (Циклічний таймер):** Робота за заданими інтервалами (наприклад, 10 хв увімкнено, 50 хв вимкнено).
4. **Профілі "День/Ніч":** Можливість задати різні параметри автоматики для денного та нічного часу. Перемикання відбувається автоматично за годинником.

## 📲 Інтеграція з Telegram
- **Повноцінний Дашборд:** Отримання статусу однією командою.
- **MiniApp:** Вбудований веб-додаток прямо в Telegram для візуального керування.
- **Сповіщення:**
    - Тривога при виході температури за межі.
    - Сповіщення про зміну статусу реле.
    - Звіт про запуск пристрою та стан живлення.
- **Керування:** Зручне меню команд та інтерактивні Inline-кнопки.

## 🌐 Веб-інтерфейс (Panel)
- **Сучасний дизайн:** Glassmorphism інтерфейс з анімаціями та темною темою.
- **Динамічні графіки:** Візуалізація історії показників за сьогодні, вчора або довільний період.
- **Журнал подій:** Детальний лог всіх дій системи (перемикання реле, зміни профілів, тривоги).
- **Менеджер файлів:** Перегляд, завантаження та видалення файлів LittleFS безпосередньо через браузер.

## 💾 Логування та Пам'ять
- **Smart Logging:** Запис даних у форматі NDJSON із розбиттям по днях (`/log/YYYY-MM-DD.ndjson`).
- **Авто-ротація:** Система автоматично видаляє найстаріші логи, коли вільне місце на LittleFS стає менше 100 КБ.
- **Бекап:** Можливість скачати всі налаштування та історію логів одним архівом `.tar` локально.

## ⚡ Системні можливості
- **OTA Оновлення:** Бездротове оновлення прошивки та веб-файлів напряму з GitHub або через завантаження файлу.
- **Power Monitor:** Відстеження наявності 220В та сповіщення про зникнення/появу живлення.
- **NTP Синхронізація:** Точний час із мережі Інтернет для роботи графіків та профілів.
- **OLED Дисплей:** Локальне відображення IP-адреси, стану WiFi, сенсорів та версії при старті.
